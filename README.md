# social-network-recommendation

Python project for data cleaning \& mutual friends



\## Overview



This project demonstrates how social network data can be processed using Python.

It includes data conversion, data cleaning, and a mutual friends based

"People You May Know" recommendation system.



\## Project Modules



\### 1. Data Conversion

\- Converts `.txt` files containing JSON into `.json`

\- Uses Python file handling and json module



\### 2. Data Cleaning

\- Removes users with empty names

\- Removes duplicate pages

\- Outputs cleaned JSON data



\### 3. People You May Know

\- Suggests friends using mutual connections

\- Uses sets and dictionaries

\- Sorts recommendations by relevance
